# Scope3 CO2 サプライチェーン可視化デモ

hyde + argo + plat によるプライバシー保護型排出量証明のインタラクティブデモ。

## 3幕構成

1. **Problem** — サプライヤーがCO2排出量を出せない現実を再現
2. **Solution** — 3つの壁を3つの技術（ZKP / TEE / FHE）で突破
3. **Result** — 実数値なしでScope3を集計できた結果

## 起動

```bash
npm install
npm run dev
```

## 技術スタック

- React + Vite
- Tailwind CSS v4
- 外部ライブラリなし
