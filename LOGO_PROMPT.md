# ValueCycleCode ロゴ生成指示書

以下の指示に従い、「ValueCycleCode」のロゴを生成してください。

## コンセプト

ValueCycleCodeは、サプライチェーン上のデータ（CO₂排出量、取引価格、農薬使用量など）を**暗号化したまま集計・証明**できるプラットフォームです。

キーワード：
- **価値（Value）の循環（Cycle）を、暗号（Code）で守る**
- 見せずに証明する
- サプライチェーンの透明性とプライバシーの両立

## デザイン要件

### 形状
- サプライチェーンの「循環」を表す円環またはループ形状
- 円環の中または周囲に鍵・ロックの要素を1つ入れる（暗号を象徴）
- シンプルで、favicon（16x16px）に縮小しても判別できること

### カラー
3つの技術モジュールのカラーを組み合わせる：
- hyde（信頼基盤）: **#378ADD**（ブルー）
- argo（証明）: **#1D9E75**（ティール/グリーン）
- plat（暗号集計）: **#7F77DD**（パープル）

背景は白または透過。
グラデーションを使う場合は、上記3色を左→右または循環方向に配置。

### テキスト
- 「ValueCycleCode」のワードマークを含む
- フォント: モダンなサンセリフ体（Geist, Inter, SF Pro 系統のイメージ）
- 「Value」「Cycle」「Code」のそれぞれを上記3色で色分けしてもよい：
  - Value → #378ADD
  - Cycle → #1D9E75
  - Code → #7F77DD

### スタイル
- フラットデザイン（影・グラデーションは最小限）
- テック企業のロゴのようなクリーンさ
- 日本の食品サプライチェーンの文脈で使うが、和風にしすぎない
- 農業や食品の要素（葉、野菜など）は入れない — これは暗号技術のプロダクト

## 生成バリエーション

以下の3パターンを生成してください：

1. **アイコン + ワードマーク（横並び）** — ヘッダー用
2. **アイコンのみ（正方形）** — favicon / アプリアイコン用
3. **縦積み（アイコン上 + テキスト下）** — スライド表紙用

## 背景

- 白背景バージョン
- 透過背景バージョン（PNG）

## サイズ

- 高解像度（1024x1024以上）で生成
- 後からリサイズして使用する

## NG事項

- 3D表現は避ける
- 文字の装飾（影、縁取り）は避ける
- ブロックチェーンのノードグラフ（よくある六角形の連結図）は避ける — 差別化のため
- 鍵穴モチーフは使わない（hyde単体のロゴと被る）
